Angular-JS PubNub Chat
=========================

WIP

# Running

1. `git clone`
1. `cd pubnub-angular-js`
1. `gem install haml`
1. `gem install compass`
1. `npm install`
1. `bower install`
1. `grunt server`
1. navigate browser to http://localhost:9000/

# Vagrant
1. `gem install berkshelf`
1. `berks install`
1. `vagrant up`
